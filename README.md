This repository contains two informational web projects developed as static websites.

The first project is a website about the National Artisanal Chorizo Festival in Ataliva, featuring event history, schedule, location, ticket information, contacts, a photo gallery, and a section to apply for Chorizo Queen. --> https://fiesta-del-chorizo.netlify.app/

The second project is a Virtual Library, organized by literary genres such as science fiction, fantasy, and mystery, where users can explore and read different books. --> https://bibliotecavirtual-bsv.netlify.app/
